# Skill Manager

Manages the complete lifecycle of Claude Skills - creating, updating, maintaining, and versioning.

## Purpose

This meta-skill provides expertise in:
- Creating new skills from documentation or knowledge
- Updating existing skills with new information
- Maintaining skill quality and consistency
- Versioning and tracking skill changes
- Following best practices for skill structure

## When to Use

Ask Claude to use this skill when you:
- Want to create a new skill for a framework or tool
- Need to update an existing skill
- Want to improve skill organization
- Need guidance on skill structure
- Want to validate your skills

## Key Features

### Skill Creation
- Templates for different skill types
- Guidance on trigger descriptions
- Content organization strategies
- Supporting file recommendations

### Skill Updates
- Update detection and planning
- Version tracking approaches
- Content refresh workflows
- Validation checklists

### Maintenance
- Quality standards
- Refactoring large skills
- Managing skill overlap
- Regular maintenance schedules

## Quick Start

**Create a new skill:**
```
User: "Create a skill for pytest testing patterns"
Claude: [Uses skill-manager to guide creation]
```

**Update existing skill:**
```
User: "Update the textual skill for version 0.50"
Claude: [Uses skill-manager to plan and implement updates]
```
