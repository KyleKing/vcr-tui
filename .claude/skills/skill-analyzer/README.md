# Skill Analyzer

Analyzes codebases to identify missing skills and recommend new ones based on project characteristics.

## Purpose

This meta-skill provides expertise in:
- Detecting frameworks and tools used in a project
- Identifying repeated patterns and conventions
- Discovering knowledge gaps in current skills
- Recommending high-value skills to create
- Prioritizing skill development

## When to Use

Ask Claude to use this skill when you:
- Want to know what skills would benefit your project
- Need an audit of current skill coverage
- Are starting with Claude Skills
- Want to prioritize skill creation
- Need to identify knowledge gaps

## Key Features

### Automated Detection
- Framework and tool discovery
- Dependency analysis
- Pattern recognition
- Convention identification

### Gap Analysis
- Compare needs vs. existing skills
- Identify missing coverage
- Find outdated content

### Recommendations
- Prioritized skill suggestions
- Impact/effort scoring
- Detailed justifications
- Implementation roadmap

## Analysis Output

Provides structured reports with:
- Current skills inventory
- High/medium/low priority recommendations
- Pattern analysis
- Next steps

## Quick Start

**Analyze your project:**
```
User: "What skills should I create for this project?"
Claude: [Uses skill-analyzer to examine codebase and recommend]
```

**Audit existing skills:**
```
User: "Are there gaps in my current skills?"
Claude: [Analyzes project vs. skills and reports gaps]
```
